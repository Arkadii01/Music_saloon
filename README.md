## Программный модуль "Музыкальный салон"

Данный модуль предназначен для любителей музыки, а также для проведения вечеринок.

# Основной функционал модуля:
- просмотр каталога музыки (с наличием поиска);
- воспроизведение музыки (пауза, перемотка, повтор);
- рекомендации (на основе выбранного жанра);
- оценка музыки (по 5-бальной шкале);
- шифрование данных (возможно);
- создание аккаунта (для сохранения плейлистов и предспочтений у пользователя).
